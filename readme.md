# WORKSHOP REPROGRAMA FRONT END

## ASSUNTOS ABORDADOS:

* FUNDAMENTOS DE HTML
* FUNDAMENTOS DE CSS
* ELABORAÇÃO DE MINI PROJETO - PÁGINA NA WEB PERFIL/CV DA ALUNA


### Passo a passo do projeto:

1. Abrir arquivo Index (html)
2. Estruturei o Index (head e body)
3. Abrir arquivo Stylo (css)
4. Adicionei as fotos que seriam utilizadas no projeto
5. No head referenciei o link do css
6. Ainda no head criei o título através da tag <title>
7. No Body, separei por temas Banner, Sobre, Link e Redes sociais
8. Coloquei as tags div e criei classes para futuras utilizações no arquivo css
9. Inseri a Foto para o Banner
10. Utilizei as tags <h> e <p> para inserir textos
11. Inseri o link do CV através da tag href e para abrir em outra página a tag _blank
12. Na parte redes sociais inseri imagens relacionados as redes sociais
13. E inseri os links de cada rede social representada.
14. Encerrei com "Copyright"
15. No arquivo css utilizei as classes criadas no html para estilizar o site
16. Utilizei as tags para deixar o site responsivo
16. Salvei, testei e conferi as funcionalidades do site
